# Cite Agent - Quick Start for Professors

## Installation (2 minutes)

1. **Download** `CiteAgentSetup-1.4.8.exe`
2. **Double-click** the installer
3. **Click Next** through the prompts
4. **Done!** A desktop shortcut appears

## First Launch

1. **Double-click** the "Cite Agent" desktop icon
2. The setup wizard launches automatically
3. Enter your **email** and **password**
4. You're ready to research!

## Basic Usage

Open any terminal (Command Prompt, PowerShell, or Windows Terminal) and type:

```
cite-agent
```

### Example Research Queries

```
Find recent papers on transformer architectures

Summarize the key findings from attention mechanism research

What are the citation trends for BERT in 2023?

Compare methodologies in recent NLP papers
```

### Quick Commands

| Command | What it does |
|---------|--------------|
| `cite-agent` | Start interactive chat |
| `cite-agent "your question"` | Quick single query |
| `cite-agent --library` | View saved papers |
| `cite-agent --history` | See past queries |
| `cite-agent --export-bibtex` | Export citations |

## Tips for Researchers

- **Save papers**: When you find useful papers, they're automatically added to your library
- **Export citations**: Use `--export-bibtex` for LaTeX or `--export-markdown` for notes
- **Cache benefit**: Repeated queries are instant (cached locally)
- **Context aware**: The AI remembers your conversation context
